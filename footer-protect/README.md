# @cybernix/footer-protect

Small React Footer component packaged for npm.

Usage (CommonJS):

```js
const React = require("react");
const { Footer } = require("@cybernix/footer-protect");

// Render <Footer /> in your React app
```

Publish (public npm):

```powershell
cd footer-protect
npm login
npm publish --access public
```

If you set `publishConfig.access` to `public`, you can omit the `--access` flag.
